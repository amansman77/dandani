## 📋 출시 전 필수 체크리스트

### 1. 앱 정보 설정

#### Xcode 프로젝트 설정
- [ ] **Display Name**: `단단이` 확인
- [ ] **Bundle Identifier**: `com.yetimates.dandani` 확인
- [ ] **Version**: `0.1.0` 확인
- [ ] **Build**: `1` 확인
- [ ] **Minimum iOS Version**: `15.0` 확인

#### App Store Connect 설정
- [ ] App Store Connect에 앱 등록
- [ ] Bundle ID 등록 확인
- [ ] 앱 이름: `단단이`
- [ ] 기본 언어: 한국어
- [ ] 카테고리 선택 (건강 및 피트니스, 라이프스타일 등)

### 2. 앱 아이콘 및 스플래시 스크린

#### 앱 아이콘
- [ ] 1024x1024 PNG 아이콘 준비
- [ ] Xcode에서 Assets.xcassets > AppIcon에 추가
- [ ] 모든 크기 자동 생성 확인

#### 스플래시 스크린
- [ ] LaunchScreen.storyboard 확인
- [ ] 배경색: `#faf5e9` 확인
- [ ] 로고 또는 텍스트 추가 (선택사항)

### 3. 권한 및 개인정보 보호

#### Info.plist 권한 설명
- [ ] 인터넷 연결 권한 (자동)
- [ ] 추가 권한이 필요한 경우 설명 추가:
  ```xml
  <key>NSPhotoLibraryUsageDescription</key>
  <string>사진을 선택하기 위해 권한이 필요합니다</string>
  ```

#### App Privacy (App Store Connect)
- [ ] 개인정보 수집 여부 확인
- [ ] 현재: 사용자 ID (로컬 스토리지, 익명)
- [ ] 개인정보 보호 정책 작성 (필요시)

### 4. 스토어 리스팅

#### 필수 정보
- [ ] 앱 이름: `단단이`
- [ ] 부제목 (선택사항)
- [ ] 설명 (최대 4000자)
- [ ] 키워드 (최대 100자)
- [ ] 지원 URL
- [ ] 마케팅 URL (선택사항)
- [ ] 개인정보 보호 정책 URL (필요시)

#### 스크린샷
- [ ] iPhone 6.7" (iPhone 14 Pro Max 등) - 필수
- [ ] iPhone 6.5" (iPhone 11 Pro Max 등) - 선택
- [ ] iPhone 5.5" (iPhone 8 Plus 등) - 선택
- [ ] 최소 3개, 최대 10개

#### 앱 미리보기 비디오 (선택사항)
- [ ] 15-30초 비디오
- [ ] 앱 주요 기능 소개

### 5. 빌드 및 Archive

#### Archive 생성
1. Xcode에서 **Product > Archive**
2. Archive 완료 대기
3. Organizer 창 확인

#### Archive 검증
- [ ] Archive 검증 성공
- [ ] 서명 확인
- [ ] 의존성 확인

### 6. TestFlight 배포 (권장)

#### 내부 테스터
- [ ] 내부 테스터 추가 (최대 100명)
- [ ] 빌드 업로드
- [ ] 테스터 초대

#### 외부 테스터 (선택사항)
- [ ] 외부 테스터 그룹 생성
- [ ] 테스터 추가 (최대 10,000명)
- [ ] 베타 앱 검토 제출 (필요시)

### 7. App Store 제출

#### 제출 전 최종 확인
- [ ] 모든 스크린샷 업로드
- [ ] 앱 설명 완성
- [ ] 키워드 설정
- [ ] 지원 URL 확인
- [ ] 가격 설정 (무료/유료)
- [ ] 가용성 국가 선택

#### 제출
1. App Store Connect에서 **Submit for Review** 클릭
2. Export Compliance 확인
3. Content Rights 확인
4. Advertising Identifier 확인
5. 제출 완료

### 8. 검토 대기

- [ ] 제출 확인 이메일 수신
- [ ] "Waiting for Review" 상태 확인
- [ ] 검토 완료 대기 (보통 1-3일)
