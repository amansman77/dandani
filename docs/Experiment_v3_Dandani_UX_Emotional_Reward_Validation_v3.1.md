# 📄 **Experiment v3 — UX & Emotional Reward Validation**

**Document Version:** v3.1
**Experiment Version:** v3
**Period:** 2025.12 ~
**Owner:** Product Manager — 황호성
**Project:** Dandani (단단이)

## 1. Overview

Experiment v3는 다음 가설을 검증한다:

> “정제된 UI/UX와 감정적 보상 구조는 사용자의 실천 완료율과 재방문율을 높인다.”

Experiment v2에서 확인된 핵심 문제는 **행동 전환 부족**과 **Retention 부재**였다.
또한 최근 2주간 PM 본인이 직접 사용하며(Case: Self-Dogfooding), **UX·문구·보상 강화가 실천 행동에 긍정적 영향을 준다는 정성적 증거**가 확보되었다.

v3의 목표는 이 정성적 증거를 **정량적 실험으로 검증**하는 것이다.

## 2. Background

### 2.1 Experiment v2 Failure Summary

Experiment v2는 다음 가설을 검증했다:

> “사용자가 공감하는 짧은 실천을 선택하면 행동과 잔존으로 이어진다.”

결과는 아래와 같다:

| 항목             | 성과                | 판단       |
| -------------- | ----------------- | -------- |
| CTR            | 5.03% (Excellent) | 공감 발생    |
| 챌린지 선택률        | 낮음                | 선택 전환 실패 |
| 실천 완료율 (PCR)   | 10~20% 수준         | 행동 전환 실패 |
| Day2 Retention | 4.2%              | 유지 실패    |

### 핵심 문제 요약

1. **보상 구조 부족 → 실천 행동을 유도하지 못함**
2. **UX 혼란 → 진행률/날짜 불신 → 지속 의지 약화**
3. **감정적 루프 없음 → 사용자가 의미를 느끼지 못함**
4. **리마인더 없음 → 재방문 동기 부재**

### 2.2 Self-Dogfooding Insight (2주간 직접 사용 결과)

PM 본인이 단단이를 실제로 2주간 사용한 결과, 다음 변화가 즉시 포착되었다:

1. **문구가 좋아지니 챌린지 몰입감 증가**
2. **완료 UI(애니메이션/색 변화)가 강화되니 완료를 누르고 싶어짐**
3. **진행률·날짜 정확성 → 신뢰도 상승**
4. **전체적 UX 매끄러움 → 서비스 만족도 상승**

이 경험은 v2의 실패 분석과 정확히 일치하며, v3 가설의 기반이 된다.

## 3. Hypothesis

### 3.1 Primary Hypothesis (H1)

> H1 — “정교한 UX와 강화된 감정적 보상은 실천 완료율과 재방문율을 증가시킨다.”

### 3.2 Supporting Hypotheses

— 사용자가 선택해야 하는 카드의 강조 효과는 클릭률을 증가시킨다.
— 감정적으로 설계된 챌린지 문구는 챌린지 지속 일수(도달율)를 증가시킨다.
— 정확한 날짜·진행률·상태 계산은 사용자의 신뢰를 높여 이탈률을 감소시킨다.

## 4. Experiment Objective

Experiment v3의 목적은 다음 두 가지다:

1. **UX 개선 → 행동 전환률(PCR) 상승을 검증**

## 5. Success Metrics

## 5.1 Behavior Conversion Metrics

| Metric                   | 목표       |
| ------------------------ | -------- |
| **1일차 실천 완료율(PCR-Day1)** | ≥ 30%    |
| **3일차 실천 완료율(PCR-Day3)** | ≥ 20%    |
| **완료 버튼 클릭률**            | v2 대비 2배 |

## 5.2 Retention Metrics

| Metric              | 목표    |
| ------------------- | ----- |
| **Day2 Retention**  | ≥ 10% |
| **Day3 Retention**  | ≥ 8%  |
| **7-Day Retention** | ≥ 5%  |

## 5.3 Challenge Progress Metrics

| Metric                        | 목표         |
| ----------------------------- | ---------- |
| **3일차 도달율(Completion Reach)** | ≥ 20%      |
| **중도 이탈률 감소**                 | v2 대비 -30% |

## 6. Experiment Scope

## 6.1 포함 범위

* "아침 선언"을 포함한 챌린지 문구 업데이트
* 진행률·날짜·상태 계산의 정합성 확보
* 실천 완료 카드의 시각적 보상(색·애니메이션) 강화
* AlertModal 도입 → 피드백 일관성 증가
* 추천/인기 챌린지 UI 정렬
* UX 흐름 (선택 → 실천 → 완료) 전체 안정화

## 6.2 제외 범위

* Push Notification (다음 실험 단계)
* AI 기반 감정 코칭
* 장기 챌린지 설계
* B2B/커뮤니티 기능

## 7. Experiment Design

## 7.1 Pre-Experiment

* v3 기능 배포
* 이벤트 로깅 체계 정비
* 픽셀 및 행동 로그 세부 데이터포인트 정의

## 7.2 During Experiment

* 신규 사용자 3~7일간 관찰
* 행동 퍼널: 선택 → 실천 → 완료 → 다음날 방문

## 7.3 Post-Experiment

* 퍼널 데이터 분석 (v2 대비)
* 감정적 보상 요소의 기여도 분석
* “완료 버튼 클릭 전후 행동” 분석
* 문구 효과 검증(A/B형 가능)

## 8. Expected Outcome

Experiment v3가 성공한다면…

1. 실천 완료율이 명확하게 증가한다.
2. 감정 기반 UI가 행동 전환에 실질적으로 기여한다.
3. 재방문 지표가 개선되어 “루틴 형성 가능성”이 보인다.
4. 단단이의 핵심 가치:
   **“상황이 흔들어도 중심을 잃지 않게 하는 감정 루프”**
   를 증명할 수 있다.

실패하더라도 다음 단계(v4: 리마인더·습관 루프)에 대한 확신을 제공한다.

## 9. Risks & Mitigation

| Risk                               | 대응 전략                |
| ---------------------------------- | -------------------- |
| UX 개선에도 불구하고 행동 전환이 크게 오르지 않을 수 있음 | v4에서 리마인더·루틴 설계 추가   |
| 문구 효과가 사용자마다 상이                    | 문구 A/B 테스트 도입        |
| 방문성 문제                             | 홈 화면 바로가기(PWA) 실험    |
| 초기 유입량 부족                          | 타겟팅 강화, CTR 기반 광고 운영 |

## 10. Decision Rule

Experiment v3는 아래 조건 중 **하나라도 충족하면 성공(Success)** 으로 본다:

1. 실천 완료율 ≥ 30%
2. Day2 Retention ≥ 10%
3. 감정적 피드백에 대한 사용자 반응 증가
4. v2 대비 퍼널 개선이 명확하게 확인됨

충족 실패 시 → **v4 실험(리마인더 중심 루프 실험)로 즉시 이동**

## 11. Next Step (Experiment v4 Preview)

* “어제의 나와 연결되는” 감정 코칭 문구 실험
* PWA 기반 접근성 향상
* 장기 챌린지 설계

## 12. Appendix — Why Experiment v3 Matters

v3는 단단이 서비스가 단순한 "문구 제공 챌린지"가 아니라
**‘감정 회복 루프’를 설계하는 서비스**임을 증명하는 결정적 실험이다.

이 실험이 성공하면:

* UX·감정 기반 설계의 ROI가 확인되고
* 장기적인 제품 방향(Emotion Loop Platform)이 강화되며
* 단단이의 브랜드 아이덴티티도 더욱 명확해진다.
